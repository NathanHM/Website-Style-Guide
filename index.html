  • Montserrat is to be used for titles and headings.

  • Arial is to be used for body text.

  • Source Code Pro is to be used for code.

  • Inconsolata is to be used for small print, i.e. footer.

Text Styles

H1: Title Card:

  • Font Size: 100px

  • Font Styling: None

  • Dark Theme Colour: #ffffff

  • Light Theme Colour: #000000

H2: Subtitle Card:

  • Font Size: 50px

  • Font Styling: None

  • Dark Theme Colour: #ffffff

  • Light Theme Colour: #000000

H3: Headings:

  • Font Size: 30px

  • Font Styling: Underlined

  • Dark Theme Colour: #ffffff

  • Light Theme Colour: #000000

H4: Subheadings:

  • Font Size: 25px

  • Font Styling: Underlined

  • Dark Theme Colour: #ffffff

  • Light Theme Colour: #000000

P: Paragraph Text:

  • Font Size: 18px

  • Font Styling: 1px letter spacing

  • Dark Theme Colour: #ffffff

  • Light Theme Colour: #000000

A: Links:

  • Font Size: Inherited

  • Font Styling: Inherited, underlined on hover, colour shift on click

  • Colour: #5900FF

  • Active Dark Theme Colour: #ffffff

  • Active Light Theme Colour: #000000
